This movie API will provide users with acess to imformation about various directors, movies and genres. Users will also be able to sign up , update personal information , and create a list of favorite movies.

Data Return URL HTTP Method Paramaters Response
Returns a list of all movies /movies GET None A JSON object of all the movies as an array
Returns the data of a single movie by title /movies/:title GET :title A JSON object showing data on a single movie
Returns data about a genre by name movies/genre/:genreName GET :genreName A JSON object of a genre
Returns data about a director by name movies/directors/:directorName GET :directorName A JSON object of a director
Returns a list of all users /users GET none A JSON object of all users as an array
Returns data about a specific user by name /users/:userName GET :userName A JSON object of data on a single user by name
Allows new users to register /users POST none A JSON of the new user and their details
Allows users to update their info /users/:id PUT :id A JSON of the users updated info
Allows users to add a movie to their favourites /users/:id/:movieTitle POST :id, :movieTitle Text confirming a movie had been added to their favourites
Allows users to remove a movie to their favourites /users/:id/movieTitle DELETE :id, :movieTitle Text confirming a movie had been removed from their favourites
Allows users to deregister /users/:id DELETE :id Text confirming a user has been removed
